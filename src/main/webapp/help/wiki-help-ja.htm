テキスト整形のルール

段落

  • 空行が段落の区切りとなります。テキストの改行は表示に反映されません。
  • 段落内に改行を挿入したい場合は、行末に ~ を書きます。

文字装飾

-'''太字'''
-''イタリック''
-__打ち消し線__
表示:
  • 太字
  • イタリック
  • 打ち消し線

箇条書き

-項目 1
--項目 1.1
-項目 2

+項目 1
++項目 1.1
+項目 2
表示:
  • 項目 1
    • 項目 1.1
  • 項目 2
  1. 項目 1
    1. 項目 1.1
  2. 項目 2

用語解説

:用語1:用語の説明をここに書きます
:用語2:用語の説明をここに書きます
表示:
用語1
用語の説明をここに書きます
用語2
用語の説明をここに書きます

整形済みテキスト

行頭がスペースで始まっていると、その段落は整形済みとして扱われます。

 public static void main(String[] args) {
   System.out.println("Hello world!");
 }
表示: 
 public static void main(String[] args) {
   System.out.println("Hello world!");
 }

引用文

> を行頭に書くと、引用文が書けます。> >> >>> の3レベルがあります。

>これは引用文です。
>>入れ子の引用も可能です。
表示:
これは引用文です。
入れ子の引用も可能です。

テーブル

項目を || で区切ると、テーブルが作れます。

||Cell 1||Cell 2||Cell 3||
||Cell 4||Cell 5||Cell 6||
表示:
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6

水平線

マイナス4個(----)を行頭に書くと、水平線になります。

----
表示:

リンク

  • タグ名とURLは、自動的にリンクになります。
  • URLの対象が画像である場合は(gif|png|jpeg|jpg)、imgタグが挿入されます。
  • YouTubeビデオへのURLは、その場所に埋め込みプレーヤーが挿入されます。
知識フラグメントへのリンク: 
-#1
-fragment:1
-fragment:1:title
-fragment:1:detail
表示:
  • #1
  • fragment:1
  • Fragment Title
  • #1 Fragment Title
ラベル付きリンク: 
-[http://www.google.com/ Google]
-[fragment:1 フラグメント #1 へのリンク]
表示:
  • Google
  • フラグメント #1 へのリンク

他のフラグメントの内容を埋め込む

以下のように記述することで他のフラグメントの内容をその場所に埋め込むことができます。 

fragment:1:embed
