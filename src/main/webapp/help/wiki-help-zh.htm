文本格式规则

段落

  • 用空行来分开段落。换行符不会反映在显示的文本上。
  • 要在段落内改行时，在行末加符号 ~ 。

文字的修饰

-'''粗体'''
-''斜体''
-__删除线__
显示:
  • 粗体
  • 斜体
  • 删除线

分项写

-项目 1
--项目 1.1
-项目 2

+项目 1
++项目 1.1
+项目 2
显示:
  • 项目 1
    • 项目 1.1
  • 项目 2
  1. 项目 1
    1. 项目 1.1
  2. 项目 2

词汇表

:词汇1:在这里写词汇的说明
:词汇2:在这里写词汇的说明
显示:
词汇1
在这里写词汇的说明
词汇2
在这里写词汇的说明

预格式化文本

行头用空格开始时、接下来的段落被视为预格式化文本。

 public static void main(String[] args) {
   System.out.println("Hello world!");
 }
显示: 
 public static void main(String[] args) {
   System.out.println("Hello world!");
 }

引用文

在行头写符号 > ，表示这是引用文。> >> >>> 有三个层次。

>这是引用文。
>>可以几重引用。
显示:
这是引用文。
可以几重引用。

表格

在项目之间用符号 || 来分隔，可以做表格。

||Cell 1||Cell 2||Cell 3||
||Cell 4||Cell 5||Cell 6||
显示:
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6

水平线

在行头写减号4個(----)，是水平线。

----
显示:

链接

  • 自动地编制标签名和URL的链接。
  • URL的対象是图像的话(gif|png|jpeg|jpg)，会自动插入img的标签。
  • 如果是YouTube视频URL, 会自动插入YouTube视频阅览器代码。
知识帖子的链接: 
-#1
-fragment:1
-fragment:1:title
-fragment:1:detail
显示:
  • #1
  • fragment:1
  • Fragment Title
  • #1 Fragment Title
注释的链接 
-[http://www.baidu.com/ 百度]
-[fragment:1 帖子 #1 的链接]
显示:
  • 百度
  • 帖子 #1 的链接

嵌入其他帖子的内容

您可以按照以下的写法，在帖子里嵌入到其他帖子的内容。 

fragment:1:embed
